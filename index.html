My Portfolio

Rogue Pickings

HTML5 & CSS from homepage design comp and additional self-designed pages

Rogue Pickings is a website for a food truck company based out of San Francisco which values using whole ingredients for its recipes. As my first website with multiple pages, it allowed me to practice linking all the pages together and maintaining a consistent theme throughout. The site also features embedded google maps and reviews, a gallery page, and a chance to showcase my writing.

Jubilee Austen Portfolio

HTML5 & CSS Website

This portfolio page was the first website I created from scratch using a design comp provided by Skillcrush. It is intended for new web developer featuring a clean and simple design. It includes navigation links that will direct you to different portions of the page as well as links to a site email and social media pages.
